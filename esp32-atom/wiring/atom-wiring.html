Notes

  • All 3 signals on GPIO <32 (fast ISR register path).
  • Power Atom from spa 5V pin; 3V3 out feeds BSS138 LV.
  • 470 Ω optional — G19/22/23 aren't strap pins, no boot back-power issue.
  • ISRs pinned to core 1; WiFi on core 0 → no preemption.
  • YAML: framework: arduino required.
  • 10 µF across Atom 5V/GND; ESP32 draws ~2× the '8266, so YAML sets output_power: 12dB + power_save_mode: light to tame TX spikes.
  • Fuse the 5V tap at ~500 mA–1 A slow-blow.

Avoid

  • G33 / G32 — work but are GPIO ≥32 (can't use fast path).
  • G0 (strap/button), G34–39 (input-only).
  • G27 (RGB LED), G12 (IR) — used internally.
